# aesynx

`aesynx` is the repository for the Aesynx operating-system project.

Aesynx is planned as a Rust `no_std` research OS with explicit capabilities, per-core ownership, service queues, driver isolation, an immutable object graph, native userspace, and AI-ready telemetry from day one.

The first major goal is not a desktop OS or Unix clone. The first major goal is a serious QEMU release.
